# MedicalCard Security

## Описание проекта

Проект выполнен в рамках задания по дисциплине «Информационная безопасность».

Работа включает:
- разработку концепции защиты мобильного медицинского приложения MedicalCard;
- реализацию шифра Цезаря на языке Python для демонстрации механизма защиты пароля.

---

## Реализация шифра Цезаря

Скрипт:
- запрашивает пароль пользователя;
- шифрует его методом сдвига символов;
- расшифровывает обратно.

Ключ сдвига: 5.


## Пример работы

Исходный пароль:
MedCard2026!

Зашифрованный пароль:
(генерируется программой)

Расшифрованный пароль:
MedCard2026!
